# Tablet Countdown Timer (平板專用倒數計時器)

一個專為平板與大型顯示器設計的高質感倒數計時器網頁應用。

## 主要特色

- **極致視覺效果**：超大數字顯示，搭配隨時間變色的全螢幕呼吸霓虹燈外框。
- **自動全螢幕**：點擊開始自動進入全螢幕模式，消除瀏覽器介面干擾。
- **專業級音效**：使用 Web Audio API 合成的溫潤和弦警報，具備空間殘響 (Reverb) 效果。
- **智慧隱藏控制**：計時開始後自動隱藏按鈕，點擊螢幕任意處即可喚回控制列。
- **高自由度設定**：支援快速加時、長按微調以及鍵盤直接輸入。
- **狀態持久化**：自動儲存您的時間設定與主題偏好。

## 檔案說明

- `index.html`: 全新極致視聽版（推薦使用）。
- `平板倒數計時器.html`: 原始穩定版本。

## 使用方式

直接使用現代瀏覽器（如 Chrome, Edge, Safari）開啟 `index.html` 即可使用。
