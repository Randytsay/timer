# Tablet Countdown Timer (平板專用倒數計時器)

一個專為平板與大型顯示器設計的高質感倒數計時器網頁應用。

## 主要特色

- **極致視覺效果**：超大「Orbitron」科技感字體，搭配**經典銀色金屬 (Silver Chrome)** 質感，視覺衝擊力極強。
- **全屏全方位呼吸燈**：螢幕四邊皆有隨時間變色（青藍 → 橙 → 紅）的霓虹呼吸燈效果。
- **穩定顯示技術**：使用等寬數字 (Tabular Nums) 解決計時跳動問題，並優化了全螢幕縮放比例防止溢出。
- **自動全螢幕模式**：點擊開始自動進入全螢幕，徹底消除介面干擾。
- **自訂音效系統**：內建專業和弦音效，並支援上傳個人音樂作為自訂鬧鈴。
- **暫停即時微調**：計時期間點擊螢幕即可喚回控制列，並在暫停狀態下直接加減時間。
- **跨平台狀態儲存**：自動記憶最後一次設定的時間與音效模式。

## 檔案說明

- `index.html`: 全新極致「銀色金屬」視聽版（主入口）。
- `平板倒數計時器.html`: 原始穩定版本。

## 使用方式

直接使用現代瀏覽器開啟 `index.html`。建議在平板上使用，並點擊中間「開始」按鈕以獲得最佳全螢幕體驗。
